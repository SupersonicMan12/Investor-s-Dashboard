# Investor's Dashboard (Step 1)

This is the first project milestone: a Python-first desktop web app using FastAPI + Jinja templates.

## Run locally (one command)

```bash
./scripts/run-dev.sh
```

Open <http://127.0.0.1:8000> in your browser.

## Included in Step 1

- FastAPI app bootstrap
- Basic server-rendered dashboard page
- Static CSS styling
- Health endpoint (`/health`)
